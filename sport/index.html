Single-Arm Machine Row
Back Hard Drop Set 3 min

- Symmetry First: Always start with the left arm and match reps with the right

- Chest Glued: No torso rotation; keep sternum against the pad

- Hook Grip: Hands are just hooks; drive the elbow toward the hip

- Stability: Grip the machine frame with your free hand

Neutral-Grip Lat Pulldown
Back Medium Drop Set 2 min

- Lats Focus: Pull with elbows; avoid leaning back excessively
